# Memory

This file is where AI stores things it learns about you across conversations. Unlike CLAUDE.md (which is your instructions), this file is built up over time as patterns emerge.

You can also add things here manually. If you want AI to always remember something, write it down here.

## What goes here

- Patterns and preferences confirmed across multiple conversations
- Key decisions and the reasoning behind them
- People and relationships that keep coming up
- Solutions to problems you've solved before
- Anything you'd want a new assistant to know about you on day one

## What doesn't go here

- Temporary or session-specific context (that goes in daily notes)
- Instructions for how AI should behave (that goes in CLAUDE.md)
- Anything that duplicates what's already in CLAUDE.md
